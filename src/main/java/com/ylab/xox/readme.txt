Точки входа:
1. XoxApplication - метод main,
в наследство от предыдущих версий остался класс Main
если вдруг захочется поиграть через консоль

вход через браузер
http://localhost:8080/gameplay

Во вкладке браузера "Воспроизвести игру" выводится список всех xml и json файлов
с сохраненными геймплеями из папки "savedFiles", при клике на любой из них
на этой же странице ниже списка файлов происходит воспроизведение ранее записанной игры

Контроллер находится в папке controllers, в нем я нагородил много строк
которые хорошо бы разнести по отдельным методам но пока не успел

Файлы xml и json пишутся и читаются из папки savedFiles
файл статистики игр в папке statistics

Файлы сохранения хода игры пишем в формате gameplay_dd_MM_yyyy_HH_mm_ss.xml/json чтобы не было одинаковых имен

Классы парсеров в папке parsers, кушаем разные виды форматов ввода координат хода игрока

классы моделей в папке models (Person) Для сериализации статистики игр,
остальные для создания модели при сериализации в json/xml

Все игровые методы находятся в классе GameMethods, его наследниками являются классы:

1 - GameWithHuman -реализует все игровые методы при игре с человеком,
результаты сохраняются в объект Gameplay

2. GameFromGameplay - реализует игровые методы при воспроизведении игры из объекта gameplay получаемого из json/xml
